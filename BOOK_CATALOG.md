## Overview

This catalog provides a comprehensive index of all Python programming books available in the repository. Books are organized by language (English/Chinese) and categorized by topic and difficulty level.

### Collection Summary
- **Total Books**: 200+ titles
- **English Books**: 150+ titles in various formats (PDF, EPUB, AZW3)
- **Chinese Books**: 100+ titles primarily in PDF format
- **Topics Covered**: Web Development, Data Science, Machine Learning, GUI Programming, Testing, Security, and more

### Format Distribution
- **PDF**: Primary format for most books
- **EPUB**: Available for many titles
- **MOBI**: Kindle-compatible format for select books
- **AZW3**: Amazon format for newer titles
- **ZIP/RAR**: Source code and supplementary materials
